# UnityTest
▶ 문제: 두개의 캐릭터가 서로 자동 전투하는 미니 게임을 만들어 보시오.<br/>
    조건1) 게임을 시작하면 두 캐릭터가 자동으로 전투한다.<br/>
    조건2) 체력은 "100" 부터 시작하고 HP Bar는 머리 위에 그려줘야 한다.<br/>
    조건3) 캐릭터는 2가지 이상의 모션을 가지고 있으며 액션에 따라서 애니메이션이 바뀌어야 한다.<br/>
    조건4) HP가 먼저 "0" 이 되는 캐릭터가 게임에서 패배한다.<br/>
    조건5) 게임 플레이 제한 시간은 총 60초이며 이 시간이 초과되면 HP가 많이 남은 캐릭터가 승리한다.<br/>
    조건6) 누가 이겼는지 출력해준다.<br/>
